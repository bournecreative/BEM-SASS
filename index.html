Responsive Grid System

Check out the scss files in this project to see how we leverage scss to build out a robust rule set using a functional css approach.

The Grid System is a 12 column grid with 4 break points (1200px, 1025, 768 and 768). Go ahead test it out! Above 1200px, the columns span 4 columns. Above 1025px, each column spans 2 columns. Above 768px, each each columns spans 6 solumns. Above 320px, each column spans the full 12 columns.
